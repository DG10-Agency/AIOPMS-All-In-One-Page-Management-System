# ArtitechCore Plugin Screenshots

## Screenshot Naming Convention

For WordPress.org plugin submission, place PNG screenshots in this folder with these exact names:

| Filename | Tab to Capture |
|----------|----------------|
| `screenshot-1.png` | **Manual Page Creation** - Show the text input area with hierarchy syntax |
| `screenshot-2.png` | **CSV Upload** - Show the file upload interface |
| `screenshot-3.png` | **AI Generation** - Show the form with business type fields |
| `screenshot-4.png` | **Schema Generator** - Show schema type selection and preview |
| `screenshot-5.png` | **Menu Generator** - Show the menu type cards grid |
| `screenshot-6.png` | **Page Hierarchy** - Show the interactive tree view |
| `screenshot-7.png` | **Keyword Analysis** - Show analysis results with table |
| `screenshot-8.png` | **Custom Post Types** - Show CPT list or creation form |
| `screenshot-9.png` | **Settings** - Show AI provider settings |

## WordPress.org Requirements

- **Format**: PNG (recommended) or JPG
- **Size**: 1544x500 to 1544x940 pixels recommended (4:3 or 16:9 ratio works well)
- **Resolution**: High quality, retina-ready if possible
- **Content**: Show actual plugin UI with realistic data

## How to Capture

1. Navigate to WordPress Admin → ArtitechCore
2. Go to each tab and capture the full interface
3. Use Windows Snipping Tool (Win+Shift+S) or similar
4. Save with the exact filenames above
5. Place in this `assets/screenshots/` folder
