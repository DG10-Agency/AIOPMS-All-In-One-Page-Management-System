# Changelog

All notable changes to the ArtitechCore plugin will be documented in this file.

## [1.0.0] 2026-03-17

### 🎉 Initial Release
- **Complete UI Redesign**: Modern DG10 Agency branding and responsive interface.
- **Advanced AI Integration**: Support for OpenAI, Gemini, and DeepSeek content generation.
- **Schema & SEO Management**: Automatic Schema.org markup with intelligent detection.
- **CPT & Taxonomy Manager**: Dynamic custom post type creation and metadata management.
- **Menu Architecture**: Automatic navigation system generation.
- **Security & Performance**: Enhanced CSRF protection, rate limiting, and WordPress object caching.
